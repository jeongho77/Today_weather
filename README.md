## 프로젝트 소개
* Weather API - OpenWeatherMap를 사용하여 날씨 데이터를 받아왔습니다.
* 울산의 지역별로 날씨를 조회 할 수 있습니다.(중구,남구,동구,북구,울주군)
* 최대 4~5일의 날씨를 알 수 있습니다.
* 현재 기온에 맞는 옷을 추천 받을 수 있습니다.
